## How Tools Work

Each tool:

1. **Receives parameters** - JSON input from the MCP client
2. **Validates input** - Checks required fields and formats
3. **Performs operation** - Reads/writes vault files directly
4. **Returns results** - JSON response with data or confirmation

### Example Tool Call

When you ask "show me notes tagged with #project", the AI:

1. Selects the `search-by-tags` tool
2. Sends `{"tags": "project"}`
3. Receives matching notes with paths and context
4. Formats the response for you

## Tool Naming Convention

Tools use kebab-case names that describe their action:

- `read-note` / `write-note` - Single note operations
- `read-notes` - Batch read (plural)
- `list-*` - Enumerate items
- `search-*` - Find by criteria
- `get-*` - Retrieve specific data
- `set-*` / `add-*` - Modify data
- `bulk-*` - Batch modifications
