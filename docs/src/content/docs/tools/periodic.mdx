Periodic notes help you maintain regular journaling, reviews, and planning across different time scales.

## Daily Notes

### daily-note

Get or create a daily note.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `date` | string | No | Date for the note (default: today) |
| `folder` | string | No | Folder for daily notes (default: 'daily') |
| `format` | string | No | Date format for filename (default: '2006-01-02') |
| `create` | boolean | No | Create if missing (default: true) |

**Date formats supported:**
- `2024-01-15` (YYYY-MM-DD)
- `01-15-2024` (MM-DD-YYYY)
- `Jan 15, 2024`
- `yesterday`, `today`, `tomorrow`

**Example prompts:**
- "Open today's daily note"
- "Create a daily note for yesterday"
- "Show me my daily note from January 15th"

## Weekly Notes

### weekly-note

Get or create a weekly note.

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `date` | string | No | Date within the week (default: this week) |
| `folder` | string | No | Folder for weekly notes (default: 'weekly') |
| `format` | string | No | Filename format (default: '2006-W02' for YYYY-Www) |
| `create` | boolean | No | Create if missing (default: true) |

Any date within a week will access that week's note.

**Example prompts:**
- "Open this week's note"
- "Create a weekly note for last week"
- "Show the weekly note containing January 15th"

## Custom Formats

### Go Time Format Reference

obsidian-go-mcp uses Go's time format with reference date `Mon Jan 2 15:04:05 MST 2006`:

| Pattern | Meaning | Example |
|---------|---------|---------|
| `2006` | 4-digit year | 2024 |
| `06` | 2-digit year | 24 |
| `01` | Zero-padded month | 03 |
| `1` | Month | 3 |
| `Jan` | Month abbreviation | Mar |
| `02` | Zero-padded day | 05 |
| `2` | Day | 5 |
| `Monday` | Day name | Tuesday |
| `Mon` | Day abbreviation | Tue |

**Custom format examples:**
- `2006-01-02` → `2024-03-15`
- `January 2006` → `March 2024`
- `2006-W02` → `2024-W11` (ISO week)
