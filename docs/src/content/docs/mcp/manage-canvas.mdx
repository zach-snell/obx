---
title: manage-canvas
description: Create notes and edges within Obsidian Canvas interface.
---

The `manage-canvas` MCP tool can parse and edit `.canvas` JSON files interactively.

## Actions

- `list`: Find all canvases in the vault.
- `read`: Get the parsed JSON graph nodes and edges.
- `create`: Generate a blank canvas setup.
- `add-node`: Append a note or text block somewhere on the 2D plane.
- `add-edge`: Draw a mathematical connection line between two nodes.
