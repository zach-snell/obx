---
title: manage-periodic-notes
description: Fetch or create daily, weekly, monthly, quarterly, or yearly notes.
---

The `manage-periodic-notes` MCP tool is tightly integrated with Obsidian's Periodic Notes workflow.

## Actions

- `daily`: Creates or fetches the Daily note for a date.
- `weekly`: Creates or fetches the Weekly note.
- `monthly`: Creates or fetches the Monthly note.
- `quarterly`: Creates or fetches the Quarterly note.
- `yearly`: Creates or fetches the Yearly note.
- `list-daily`: Lists chronological daily notes.
- `list-periodic`: Lists chronological generic periodic notes.
