title: manage-mocs
description: Discover existing Maps of Content or automatically generate new index files.
---

The `manage-mocs` MCP tool handles hierarchical documentation patterns inside Obsidian.

## Actions

- `discover`: Uses AI clustering to suggest parent index categories for disorganized directories.
- `generate`: Generates an intelligent Map of Content Markdown file.
- `update`: Updates an existing MOC.
- `generate-index`: Creates a literal A-Z index of a folder directory.
